API Reference


You can interact with the three available resources (tournaments, archers, and clubs) by using the GET, POST, PATCH, and DELETE HTTP methods.

The next section will explain how to send requests to the API's endpoints, and what information is expected to be sent and received.

Tournament


GET

/api/tournaments


Returns a list of all tournaments.

        
      {
          "tournaments": [
              {
                  "name": "coolTournament",
                  "id": 1,
                  "rounds": 3
              },
              {
                  "name": "radTournament",
                  "id": 2,
                  "rounds": "Not specified"
              },
              {
                  "name": "awesomeTournament",
                  "id": 3,
                  "rounds": 5
              }
          ]
      }

Query parameters:

  • id
  • name

name

: string

The name of the tournament.

id

: int

The identifier of the tournament.

rounds

: int

How many rounds the tournament will last.


POST

/api/tournaments


Adds a new tournament.

        
      {
        "code": 200,
        "message": "Tournament created successfuly."
      }

Body parameters:

name

: string

(required)

The name of the tournament.


GET

/api/tournaments/<tournament_id>


Returns information about a specific tournament.

        
      {
          "name": "coolTournament",
          "id": 1,
          "rounds": 3
      }

name

: string

The name of the tournament.

id

: int

The identifier of the tournament.

rounds

: int

How many rounds the tournament will last.


PATCH

/api/tournaments/<tournament_id>


Updates a tournament.

        
      {
        "code": 200,
        "message": "Tournament updated successfuly."
      }

Body parameters:

name

: string

(optional)

The name of the tournament.

rounds

: int

(optional)

How many rounds the tournament will last.


DELETE

/api/tournaments/<tournament_id>


Deletes a tournament.

        
      {
        "code": 200,
        "message": "Tournament deleted successfuly."
      }

Archer


GET

/api/tournaments/<tournament_id>/archers


Returns a list of all archers registered in a specific tournament.

        
      {
          "archers": [
              {
                  "name": "James",
                  "id": 3,
                  "club": "http://127.0.0.1:8000/api/clubs/coolClub",
                  "score": 3
              },
              {
                  "name": "Daniel",
                  "id": 4,
                  "club": "http://127.0.0.1:8000/api/clubs/coolClub",
                  "score": 0
              }
          ]
      }

Query parameters:

  • id
  • name
  • score
  • club

name

: string

The name of the tournament.

id

: int

The identifier of the tournament.

club

: string

Url to the club resource to which the archer belongs to.

score

: int

Current score of the archer.


POST

/api/tournaments/<tournament_id>/archers


Adds a new archer to a specific tournament.

        
      {
        "code": 200,
        "message": "Archer registered successfuly."
      }

Body parameters:

name

: string

(required)

The name of the archer.

club

: string

(required)

The club to which the archer belongs.


GET

/api/tournaments/<tournament_id>/archers/<archer_id>


Returns information about a specific archer in a specific tournament.

        
      {
          "name": "James",
          "id": 3,
          "club": "http://127.0.0.1:8000/api/clubs/coolClub",
          "score": 3
      }

name

: string

The name of the tournament.

id

: int

The identifier of the tournament.

club

: string

The club to which the archer belongs.

score

: int

Current score of the archer.


PATCH

/api/tournaments/<tournament_id>/archers/<archer_id>


Updates an archer.

        
      {
        "code": 200,
        "message": "Archer updated successfuly."
      }

Body parameters:

name

: string

(optional)

The name of the archer.

addScore

: int

(optional)

How many points were scored in a round (It's added to the archer's total score).


DELETE

/api/tournaments/<tournament_id>/archers/<archer_id>


Deletes an archer.

        
      {
        "code": 200,
        "message": "Archer deleted successfuly."
      }

Club


GET

/api/clubs


Returns a list of all clubs.

      
        {
            "clubs": [
                {
                    "name": "coolClub",
                    "firstColor": "#6391db",
                    "secondColor": "#c92c61",
                    "logo": "logo.svg"
                },
                {
                    "name": "epicClub",
                    "firstColor": "#c92c61",
                    "secondColor": "#6391db",
                    "logo": "logo.svg"
                },
                {
                    "name": "radClub",
                    "firstColor": "#6391db",
                    "secondColor": "#c92c61",
                    "logo": "logo.svg"
                }
            ]
        }

Query parameters:

  • name

name

: string

The name of the club.

firstColor

: string

Hexadecimal representation of the main color of the club.

secondColor

: string

Hexadecimal representation of the secondary color of the club.

logo

: string

Url to the logo in svg format.


GET

/api/clubs/<club_name>


Returns information about a specific club.

      
        {
            "name": "coolClub",
            "firstColor": "#6391db",
            "secondColor": "#c92c61",
            "logo": "logo.svg"
        }

name

: string

The name of the tournament.

firstColor

: string

Hexadecimal representation of the main color of the club.

secondColor

: string

Hexadecimal representation of the secondary color of the club.

logo

: string

Url to the logo in svg format.
